Create Pull Request!
By filling out the form on the left you can create Pull Request and assign reviewers with one shot!
Owner is the owner of a repository.
Repository is the name of a repository.
Head is the name of a branch that you've implemented the changes.
Base is the name of a branch that you'd like to merge into.
Reviewers are a list of user logins that you'd like to assign the review to.
Note that reviewers should be comma-separated without spaces i.e. 'user,user1'.
PR Body is optional, but you can fill it out to specify what this Pull Request will be about.
